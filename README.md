📚 Booket (북킷)
세상의 모든 취향을 예약하고(Booking), 특별한 경험을 나누는(Market) 오프라인 커뮤니티 플랫폼

1. 서비스 개요
- Booket은 사용자들이 다양한 오프라인 커뮤니티 모임을 운영하고, 개인 블로그를 통해 그 기록을 공유할 수 있는 웹 서비스입니다.

- 인원 제한이 있는 오프라인 모임의 특성을 고려하여, 사용자들이 실시간 예약을 보다 원활히 할 수 있도록 개발되었습니다.


2. 주요 기능
- OAuth 2.0 기반 소셜 로그인: 카카오, 네이버 등 소셜 계정을 통한 간편한 회원 가입 및 로그인.

- 실시간 모임 참여 예약: 인원 제한이 있는 독서 모임의 잔여 좌석을 실시간으로 관리하고 예약하는 시스템.

- 토스페이먼츠 결제 연동: 안전한 결제 API 연동을 통한 모임 참가비 결제 프로세스.

- 개인 블로그: 마크다운 에디터를 활용한 포스팅 작성 및 홍보로 커뮤니티 활성화.

- 모임별 실시간 채팅: WebSocket과 STOMP를 활용한 모임 참여자 전용 그룹 채팅방.

- 어드민 대시보드: 모임 현황 확인, 입금 관리 및 신고 내역 처리를 위한 관리자 전용 페이지.
  
(실시간 예약 정합성 확보와 대규모 트래픽 환경에서의 동시성 제어를 목표로 삼고 개발)
